第三部分：斜杠(/)命令终极解析

斜杠 (/) 命令是控制和管理 Gemini-CLI 的核心。下面是对每个命令的详细剖析。

一、 会话与历史管理 (Session & History)

/chat

管理对话会话的生命周期。

terminal
/chat save

保存当前完整的对话状态。

terminal
/chat resume

恢复一个之前保存的对话。

terminal
/chat list

列出所有已保存的会话标签。

terminal
/chat clear

清空当前对话的历史记录。

/history

查看当前会话的详细历史记录。

terminal
/history

/compress

将当前冗长的对话历史压缩成一个简洁的摘要，以节省 Token。

terminal
/compress

/clear

清空终端屏幕（视觉操作，不影响会话数据）。

terminal
/clear

/copy

将 Gemini 的上一个回答完整地复制到您的系统剪贴板中。

terminal
/copy

/restore

撤销由工具对项目所做的更改（需在启动时使用 --checkpointing 标志）。

terminal
/restore

/quit 或 /exit

退出 Gemini-CLI 交互式会话。

terminal
/quit

二、 上下文与记忆管理 (Context & Memory)

/memory

管理从 GEMINI.md 文件中加载的长期记忆。

terminal
/memory show

显示当前所有已加载的 GEMINI.md 文件内容。

terminal
/memory refresh

重新加载所有记忆文件，让更改立即生效。

terminal
/memory add ""

临时向当前会话的记忆中添加一段文本指令。

/context

查看当前完整的上下文。

terminal
/context

/context show

这是一个强大的调试工具，它会显示即将发送给模型的所有信息。

terminal
/context show

/directory

管理工作区目录。与 /path 类似，用于指定和切换 Gemini-CLI 的工作环境。

terminal
/directory

三、 工具与集成 (Tools & Integrations)

/tools

列出当前会话所有可用的工具及其状态（启用/禁用）。

terminal
/tools

/extensions

列出当前活动的扩展。扩展可以为 Gemini-CLI 增加新的功能和工具。

terminal
/extensions

/init

分析当前项目并创建一个量身定制的 GEMINI.md 文件。这是一个非常有用的命令，它会自动扫描您的代码库，识别技术栈和编码规范，并生成一个初始的记忆文件作为起点。

terminal
/init

/mcp

管理与模型上下文协议 (MCP) 服务器的连接，用于和 IDE 等外部应用集成。

terminal
/mcp

/ide

控制与 IDE 的集成功能。

terminal
/ide enable

启用集成。

terminal
/ide disable

禁用集成。

/setup-github

帮助您在当前的 Git 仓库中配置 Gemini-CLI 的 GitHub Actions 工作流。

terminal
/setup-github

/yolo

"You Only Look Once" 模式。启用后，工具执行无需手动确认，会自动批准。请谨慎使用。

terminal
/yolo

四、 配置与信息 (Configuration & Info)

/help 或 /?

显示所有可用的斜杠命令及其简要说明。

terminal
/help

/about

显示 Gemini-CLI 的版本号和相关的官方链接。

terminal
/about

/auth

管理您的身份验证方式。

terminal
/auth

/bug

引导您报告一个 Bug。

terminal
/bug

/docs

在您的默认浏览器中打开 Gemini-CLI 的官方完整文档页面。

terminal
/docs

/editor

设置当 Gemini 需要打开文件时使用的默认命令行编辑器。

terminal
/editor

/feedback

提供一个快速的渠道来提交关于 Gemini-CLI 使用体验的反馈。

terminal
/feedback

/log

显示日志文件的路径，或者直接输出最新的日志内容。

terminal
/log

/privacy

显示隐私政策声明。

terminal
/privacy

/settings

打开全局 settings.json 配置文件，或者显示其路径。

terminal
/settings

/stats

显示当前会话的详细统计数据。

terminal
/stats

/theme

更改 CLI 界面的颜色主题。

terminal
/theme

/terminal-setup

配置终端快捷键，以支持多行输入（尤其适用于 VS Code 的集成终端）。

terminal
/terminal-setup

五、 特殊模式与切换 (Special Modes & Toggles)

/corgi

切换 Corgi 模式。这是一个有趣的彩蛋功能，开启后可能会在交互中出现一些与柯基犬相关的趣味内容。

terminal
/corgi

/vim

开启或关闭 Vim 模式。对于 Vim 用户来说，这允许您在输入提示时使用熟悉的 Vim 快捷键进行文本编辑。

terminal
/vim
